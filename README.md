# Netflix Content Strategy Analysis 🎬

## Project Overview
This project provides a comprehensive analysis of Netflix's content library. Using Python and data visualization libraries, I explored the distribution of movies versus TV shows and identified the top-producing countries to understand Netflix's global business strategy.

## Core Questions Answered
* **Content Mix:** Does Netflix prioritize feature-length Movies or multi-episode TV Shows?
* **Global Reach:** Which countries are the primary hubs for Netflix content production?
* **Data Integrity:** How do we handle missing geographic and production data?

## The Data Toolbox
* **Language:** Python
* **Data Library:** Pandas (Data cleaning and manipulation)
* **Visualization:** Plotly Express (Interactive charts)
* **Database Format:** CSV (Flat-file database)

## Key Insights
1.  **Movie Dominance:** Despite the rise of "binge-watching" culture, Movies still make up the majority of the Netflix library compared to TV Shows.
2.  **Market Leaders:** The United States and India are the leading producers, showing a clear focus on the North American and South Asian markets.
3.  **Data Cleaning:** Successfully handled null values in the 'country' and 'date_added' columns to ensure 100% visualization accuracy.

## How to Run This Project
1. Clone this repository.
2. Install dependencies: `pip install pandas plotly`.
3. Run the script: `python netflix_analysis.py`.
